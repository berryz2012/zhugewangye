# 诸葛王也 - 使用指南

欢迎使用由“诸葛王也”专属定制的 AI 智能短剧分镜生成与展示工具！这是一款结合强大 AI 模型构建的综合平台，能够让你傻瓜式地完成剧本拆分成幕、角色提取、参考内容绑定，到最后的视频生成。

---

## 🚀 极速启动向导 (傻瓜式运行)

我们为您提供了一键启动的批处理脚本。即使您不懂代码，也能在几秒钟内将其运行起来。

### 环境准备：
您只需在电脑上安装好 [Node.js](https://nodejs.org/) (建议下载 LTS 稳定版)，一路点击“下一步”安装即可。

### 运行方式：
1. **Windows 用户**：双击文件夹中的 `start.bat` 文件。  
   工具会自动检测环境、安装所需依赖并启动服务。启动成功后，浏览器会自动打开或请手动访问控制台中显示的本地链接 (如 `http://localhost:3000`)。

---

## 🛠 配置你的 API

进入系统后，左侧会有完整的创作流程。在此之前，你须配置好你的专属 API。
1. 请点击侧边导航栏的第一项菜单 **【1. 全局配置】**。
2. 在这里配置即可，不需要一个个去寻找。只需配置一次即可全局生效。
3. **需要填写的接口信息**：
   - **全局 API 密钥**：你的火山引擎主 API Key（同时用于大模型及视频生成）。
   - **文本大模型 推理 ID**：可以选择 **Doubao** 或是 **DeepSeek**，用以执行文字故事拆分与提示词生成指令。
   - **视频大模型 推理 ID**：可以选择 **Seedance 2.0 (高画质)** 或 **2.0 Fast (高速度)** 的 Endpoint ID。
   （*注意：代码底层的写死备用 Key 已全部置空，保证你的密钥数据安全且私有。所有的配置都会缓存在你本地浏览器的存储中。*）

如果你不知道如何获取上述 ID，我们提供了一个弹出的 **[查看 API 接入教程]** 图文指南帮助您。

## 🎬 核心创作流程

### 第二步：资产库 (素材准备)
1. 点击左侧导航栏的【**2. 资产库**】。
2. 在这里你可以批量上传你的参考人物图片、场景图片，或者导入用作口型的音频文件 (支持mp3/wav等)。
3. 使用系统自带的人脸识别工具和画笔功能抠出人物。
4. 将重要的素材点击“★ 收藏”，后续才可在分镜里选择。

### 第三步：分镜生成 (剧本结构化)
1. 点击导航栏【**3. 分镜生成**】。
2. 你可以直接打一段小故事，或者直接导入“txt/md”格式的长篇剧本。
3. 点击生成后，AI 会自动帮你把剧本拆散成 `第1幕, 第2幕...`。
4. 在每一幕里，你可以点加号 **➕** 来关联你在第一步收藏的人物图片或音频参考。

### 第四步：视频设置 (队列化管理)
1. 点击【**4. 视频设置**】。
2. 确认你填好了全局设定并在列表上方选择了视频大模型(Seedance 2.0 / Fast)。
3. 每个独立的幕后都有个 **【生成该幕】** 的按钮，点击它！
4. **并发排队机制**：系统安全控制每次最多只有3个视频在同时生成，多出来的任务会变成【排队中（等待生成）】。一旦空闲出位置，工具就会全自动把之前排队的续上生成！
   *如果触发了官方安全识别(真人)，系统会自动无感重试3次。*

### 第五步：视频展示 (成果导出)
1. 点击【**5. 视频展示**】。
2. 在这儿，你可以看到你生成的各种长短分镜片段。
3. 点击单个视频可放大预览其效果。如果你点击了“重新生成”，这个失效视频会自动消失并重新替换成进度条。
4. 对成片满意后，点击此页面的 **【批量下载全部】** 按钮或使用单独的下载按钮。

---

如果遇到异常提示 `无法提供公网URL`, 意味着你刷新过页面把临时文件刷掉了。此时只需切回“资产库”删除掉失败的那个图片/音频，重新上传一次并在设置中重新勾选一次即可正常。
